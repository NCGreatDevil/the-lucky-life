# 好运人生 - 项目部署指南

## 一、环境安装

### 1.1 安装 Node.js（macOS 系统）

由于你的电脑没有任何 Node.js 环境，需要先安装。

**方法一：使用 Homebrew 安装（推荐）**

```bash
# 1. 如果没有安装 Homebrew，先安装
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2. 使用 Homebrew 安装 Node.js
brew install node

# 3. 验证安装
node -v
npm -v
```

**方法二：直接下载安装包**

1. 访问 Node.js 官网：https://nodejs.org/
2. 下载 macOS 安装包（LTS 版本）
3. 双击安装包，按照提示完成安装

### 1.2 配置 npm 镜像（国内加速）

```bash
# 设置淘宝镜像
npm config set registry https://registry.npmmirror.com

# 验证配置
npm config get registry
```

## 四、部署到服务器

### 4.1 使用 Nginx 部署

1. 将 `dist` 文件夹中的所有文件上传到服务器
2. 配置 Nginx：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    root /path/to/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

3. 重启 Nginx：

```bash
sudo nginx -t
sudo nginx -s reload
```

### 4.2 使用 Docker 部署

创建 `Dockerfile`：

```dockerfile
FROM nginx:alpine
COPY dist/ /usr/share/nginx/html/
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

创建 `nginx.conf`：

```nginx
server {
    listen 80;
    server_name localhost;
    root /usr/share/nginx/html;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

构建和运行：

```bash
docker build -t lucky-life .
docker run -d -p 80:80 lucky-life
```

## 六、注意事项

1. **抽签功能**：每日只能抽签一次，0点自动重置
2. **属性隐藏机制**：部分属性初始隐藏，通过抽签或事件触发后才会显示
3. **数据持久化**：当前版本数据存储在内存中，刷新页面会重置
4. **后续扩展**：可接入 localStorage 或后端 API 实现数据持久化

## 七、技术栈说明

- **Vue 3.4+** - 渐进式 JavaScript 框架，使用组合式 API
- **Vite 5+** - 下一代前端构建工具，开发体验极佳
- **Vue Router 4+** - Vue.js 官方路由管理器
- **Pinia 2+** - Vue.js 的状态管理库，Vuex 的替代方案
